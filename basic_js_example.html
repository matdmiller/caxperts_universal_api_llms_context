Universal API Example

This is a simple example of using the bundled Universal API directly in a browser.

Output will appear here...

Example Code:

import { universal_api } from './universal-api.es.js';

// Get application instance
const app = universal_api.Application.getInstance();

// Check if API is available
const available = app.available();
console.log('API Available:', available);

// If available, we could use the API to interact with the viewer
if (available) {
  // Example: Get scenes
  app.Scenes.get().then(scenes => {
    console.log('Available scenes:', scenes);
  });
}
